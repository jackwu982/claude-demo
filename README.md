# Demo

A Vue 3 personal project with an immersive dark-themed UI, built with Vite and Tailwind CSS.

## Tech Stack

- **Vue 3** — Composition API with `<script setup>`
- **Vite** — Fast dev server and build tool
- **Tailwind CSS** — Utility-first styling
- **Vue Router** — Client-side routing with page transitions
- **Three.js** — 3D rendering (Panorama page)

## Pages

| Route | Page |
|-------|------|
| `/` | Home |
| `/about` | About |
| `/contact` | Contact |
| `/panorama` | 3D Panorama viewer |

## Getting Started

```bash
pnpm install
pnpm dev
```

Build for production:

```bash
pnpm build
pnpm preview
```
